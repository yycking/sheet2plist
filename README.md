# sheet2plist
Convert Google Sheet to Property list

## Source Format (Google Sheet)
| festival:String | day:[Int] | honor:String | offering:String |
| --------------- | --------- | ------------ | --------------- |
| 初一、十五           | 1,15      | 廟神,家神        | 神明金             |
| 初二、十六           | 2,16      | 土地公          | 土地公金            |
